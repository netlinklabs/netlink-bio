# Changelog

All notable changes to Netlink.bio are documented here.
Format based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [Unreleased]

### Added
- Verification badges (Verified Profile, Verified Business, and a special recognition badge) on bio and CV pages, shown as a tappable icon next to the display name.
- Tiered link limits per subscription plan (Basic, Silver, Gold, Platinum).
- Landing page URL (slug) availability check in the page builder.
- Print and Share buttons on the CV page.
- Share button on the bio page.

### Changed
- The "Made with Netlink.bio" watermark is now automatically hidden for Gold and Platinum plans.
- Shared link previews (Open Graph images) now reflect verification status when applicable.
- CV page header layout adjusted: display name and title sizing refined, brand mark repositioned.

### Fixed
- CV contact section now shows the actual WhatsApp number instead of a generic label.
- CV shared link preview now uses a properly generated branded image instead of the raw profile photo.
